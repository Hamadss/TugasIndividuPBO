## Deskripsi Project
Student Grade System adalah program sederhana berbasis Java yang dibuat untuk mengelola data mahasiswa dan nilai mereka. Program ini dapat menyimpan informasi mahasiswa serta nilai tugas, nilai UTS, dan nilai UAS.

Program kemudian menghitung nilai rata-rata dan menentukan grade akhir berdasarkan nilai tersebut.

Project ini dibuat untuk memenuhi tugas mata kuliah **Pemrograman Berorientasi Objek (PBO)** dengan fokus pada penerapan konsep **Encapsulation**.
